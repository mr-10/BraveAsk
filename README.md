# BraveAsk 🔍

An unofficial Android client for [Brave Search Ask](https://search.brave.com/ask) — 
a clean, distraction-free app experience for Brave's AI search.

> ⚠️ This is an unofficial app. Not affiliated with or endorsed by Brave Software.
> All AI responses are powered by Brave Search's servers.

## Features
- Clean WebView wrapper around search.brave.com/ask
- Custom offline screen instead of Android's default error page
- Auto-reconnects when internet is restored
- Retry button with shake feedback
- Back navigation through history

## Built With
- Java + Android Studio
- WebView + ConnectivityManager API
- Minimum SDK: Android 7.0 (API 24)
