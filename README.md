# BraveAsk 🔍

An unofficial Android client for [Brave Search Ask](https://search.brave.com/ask) —
a clean, distraction-free native app experience for Brave's AI search.

> ⚠️ Unofficial app. Not affiliated with or endorsed by Brave Software.
> This app requires an active internet connection — all AI responses are powered by Brave Search's servers.

---

## Features

- 🌐 Clean WebView wrapper around `search.brave.com/ask`
- 📵 Custom offline screen instead of Android's default error page
- 🔄 Auto-reconnects when internet is restored
- 🔁 Retry button with shake feedback when still offline
- ◀️ Back navigation through browsing history

---

## Built With

- Java + Android Studio
- WebView + ConnectivityManager API
- Minimum SDK: Android 7.0 (API 24)
- Target SDK: Android 14 (API 34)

---

## Installation

1. Go to [Releases](https://github.com/mr-10/BraveAsk/releases)
2. Download `BraveAsk-v1.0.0.apk`
3. On your Android phone, enable **Install from unknown sources**
   > Settings → Security → Unknown sources
4. Open the APK and install
